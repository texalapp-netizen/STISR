## 简介

场景文本图像超分辨率（Scene Text Image Super-Resolution, STISR）旨在从低分辨率文本图像中恢复高分辨率、可读性强的文本图像。本仓库整理了STISR领域的主要论文及其会议/期刊官网链接。
